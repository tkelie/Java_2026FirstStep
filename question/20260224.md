## **問題1：ログイン人数カウント（難易度 C）**

### **【説明】**

あるイベントの参加者ログを表す N 行の文字列が与えられます。
ログは次の形式です：

```
<名前> <状態>
```

状態は `"IN"` または `"OUT"`。

**現在イベント会場内にいる人数を求めてください。**

### **【入力】**

```
<name1> <state1>
<name2> <state2>
...
```

### **【ルール】**

* 同じ人が何度も IN/OUT をする可能性あり
* 最後の状態が IN の人だけ人数として数える

### **【出力】**

現在会場にいる人数を整数で出力せよ。

### **【サンプル入力】**

```
alice IN
bob IN
alice OUT
carol IN
bob OUT
dave IN
```

### **【サンプル出力】**

```
2
```

（carol と dave が IN の状態）
