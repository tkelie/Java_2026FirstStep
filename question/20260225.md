## **問題2：購入合計ランキング（難易度 C〜B）**

### **【説明】**

ショッピングサイトでの購入ログが N 行与えられる。<br>
ログの形式：

```
<ユーザー名> <金額>
```

**各ユーザーが購入した合計金額を求め、
合計金額の多い順に並べて出力してください。**

### **【入力】**

```
<name1> <price1>
<name2> <price2>
...
```

### **【出力】**

```
<name> <sum_price>
```

を合計金額が多い順にすべて出力。

### **【同額の場合のルール】**

* 名前の昇順で並べる

### **【サンプル入力】**

```
alice 1200
bob 300
alice 800
carol 500
bob 700
```

### **【サンプル出力】**

```
alice 2000
bob 1000
carol 500
```
