## **問題3：連続ログイン最長時間（難易度 B）**

### **【説明】**

ユーザーごとの LOGIN / LOGOUT の時刻ログが時系列順に N 行与えられます。

形式：

```
<時刻> <状態> <ユーザー名>
```

例：

```
09:00 LOGIN alice
10:30 LOGOUT alice
```

**各ユーザーのセッション時間（ログイン→ログアウトまで）を計算し、
そのうち最も長いセッション時間を持つユーザー名と時間を出力してください。**

### **【注意】**

* ログは必ず正しい（LOGIN のあとに LOGOUT が来る）
* 分単位で計算
* 最長セッションが同じ場合は名前の昇順

### **【出力形式】**

```
<name> <minutes>
```

### **【サンプル入力】**

```
09:00 LOGIN alice
09:30 LOGIN bob
10:00 LOGOUT alice
10:30 LOGOUT bob
11:00 LOGIN bob
12:30 LOGOUT bob
```

### **【サンプル出力】**

```
bob 90
```

（11:00〜12:30 の 90分が最長）
