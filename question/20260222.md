## **🧩 問題：「文字列の集計とフィルタ」**

### **【問題文】**

あなたはログ解析ツールを作ることになりました。
入力として **N 行の文字列ログ**が与えられます。

各行は次の形式です：

```
<ユーザー名> <操作>
```

例：

```
alice LOGIN
bob LOGOUT
alice LOGIN
```

### **【やること】**

1. すべての行を読み込み
2. `LOGIN` の回数が **2 回以上のユーザー名**を
   **辞書順（アルファベット順）で昇順**に並べて出力してください
3. LOGIN が 2 回未満のユーザーは出力しません

### **【入力形式】**

```
<ユーザー名> <操作>
<ユーザー名> <操作>
...
```

### **【出力例】**

```
alice
charlie
```
