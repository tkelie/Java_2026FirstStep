## 📌 2026/02/28 の問題セット（Modern Java 練習）

### ✅ 問題 1 — **奇数のみの合計（Stream API）**

### 説明

整数 N と N 個の整数が標準入力で与えられます。
その中から **奇数だけ** を抽出して合計を出力してください。
Stream API を使う実装を意識してください。

### 入力例

```
5
1 2 3 4 5
```

### 出力例

```
9
```
