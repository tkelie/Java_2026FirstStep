
## ✅ 問題 2 — **奇数のみの合計**

### 説明

整数 N が与えられ、続く N 個の整数を読み込みます。
その中から **奇数** の値のみを合計して出力してください。

### 入力形式

```
N
a1 a2 a3 ... aN
```

### 入力例

```
5
1 2 3 4 5
```

### 出力例

```
9
```

## ✅ 問題 3 — **反転配列の表示**

### 説明

整数 N と N 個の整数が与えられます。
配列の要素を **逆順に** 出力してください。

### 入力例

```
5
10 20 30 40 50
```

### 出力例

```
50 40 30 20 10
```
