

### ✅ 問題 2 — **反転配列の表示（メソッド参照・Stream）**

### 説明

整数 N と N 個の整数が標準入力で与えられます。
配列を **逆順に表示**して出力してください。
可能なら **Stream + Collectors.toList()** を使って実装してください。

### 入力例

```
5
10 20 30 40 50
```

### 出力例

```
50 40 30 20 10
```

### ✅ 問題 3 — **Fibonacci チェッカー（Optional を活用）**

### 説明

整数 X が標準入力で入力されます。
フィボナッチ数列の中に X があれば `YES`、なければ `NO` を出力してください。
フィボナッチ数列は 0, 1, 1, 2, 3, 5,... です。
Optional を返すメソッドを実装して、それを利用する形にしてください。

### 入力例

```
13
```

### 出力例

```
YES
```

### ✅ 問題 4 — **文字列の回文判定（Modern switch 式）**

### 説明

1 行の文字列が標準入力で与えられます。
その文字列が前から読んでも後ろから読んでも同じ（回文）かを判定し、
`YES` または `NO` を出力してください。

ただし、**文字判定の条件を switch 式で実装**してください（Character クラスか判定処理を switch で扱う）。

### 入力例

```
level
```

### 出力例

```
YES
```

### ✅ 問題 5 — **日付データ集計（Date & Time API）**

### 説明

複数行の日付文字列（`yyyy-MM-dd`）が与えられます。
最も古い日付と最も新しい日付の差（日数部分）を求めて出力してください。
Date & Time API (`LocalDate`, `ChronoUnit`) を使って解いてください。

### 入力形式

```
N
date1
date2
...
dateN
```

### 入力例

```
3
2026-01-01
2026-02-15
2025-12-31
```

### 出力例

```
46
```
